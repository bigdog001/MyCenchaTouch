本代码清单内包括本书全部资源文件与全部代码文件，其中内容如下所示：
1.第一章至第十四章下各子文件夹，以及第十五章的15-1、15-2、15-3子文件夹中均包含开发阶段使用的index-debug.html页面文件（使用sencha-touch-debug.js核心脚本文件）与app.js脚本文件（开发阶段使用，未经过压缩与注释抽离处理），可拷贝到MyApp文件夹下直接运行（使用服务器端PHP脚本文件的代码示例在运行时需要为MyApp文件夹建立网站服务器、或者将MyApp文件夹下的所有子文件夹及其中文件拷贝至某个网站服务器中），MyApp文件夹下包括了所有本书中的全部图片文件以及Sencha Touch 2的Final版的所有类及组件的脚本文件(在sdk子文件夹下的src子文件夹中），运行开发阶段使用的index-debug.html脚本文件与app.js脚本文件时需要从类及组件的脚本文件中动态加载Sencha Touch 2框架的各种类与组件。

2.第一章至第十四章下各子文件夹，以及第十五章的15-1、15-2、15-3子文件夹中均包含产品正式运行阶段使用的index.html页面文件（使用sencha-touch.js核心脚本文件）与app-all.js脚本文件（产品正式运行阶段使用，经过压缩与注释抽离处理）,部分文件运行时所需要的图片文件或其他资源文件在MyApp子文件夹中，也可将index.html页面文件与app-all.js脚本文件及其他服务器端PHP脚本文件拷贝到MyApp子文件夹中直接运行。产品正式运行阶段时均从app-all.js脚本文件中动态加载应用程序所需使用的Sencha Touch 2框架的各种类与组件，不需要使用到sdk子文件夹下的src子文件夹中的任何文件。

3.第十五章的其他子文件夹与“第十六章”子文件夹中均包含创建MVC模式的应用程序时所使用的开发阶段的index.html页面文件.app.js脚本文件以及其他定义模型、视图与控制器、设备配置文件的脚本文件（在app子文件夹中），可拷贝到MyApp子文件夹下直接运行，（使用服务器端PHP脚本文件的代码示例在运行时需要为MyApp文件夹建立网站服务器、或者将MyApp文件夹下的所有子文件夹及其中文件拷贝至某个网站服务器中），需要从sdk子文件夹下的src子文件夹中动态加载Sencha Touch 2框架的各种类与组件。

4.第十五章的15-42子文件夹与“第十六章”子文件夹中均包含MVC模式的应用程序在产品正式运行阶段使用的运行时版本（在production子文件夹中），其中包括了各种图片资源文件与CSS样式文件，可在Web服务器中独立运行，不需使用sdk子文件夹下的src子文件夹中的任何文件。

另外，本代码清单中包含了用于创建全书使用到的mysql数据库中的各数据表的sql脚本文件，分别在使用到该数据表的各章子文件夹中。
